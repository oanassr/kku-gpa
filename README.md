# حاسبة المعدل الفصلي والتراكمي — جامعة الملك خالد

🔗 **النسخة المباشرة (Live):** https://oanassr.github.io/kku-gpa/

تطبيق ويب يعمل بالكامل داخل المتصفح (Client-side) لاحتساب **المعدل الفصلي والتراكمي**
لطلاب جامعة الملك خالد، مع **قراءة السجل الأكاديمي بصيغة PDF تلقائياً** واستخلاص:

- بيانات الطالب (الاسم، الرقم الجامعي، التخصص، الكلية، الدرجة).
- الحالة التراكمية السابقة (الساعات المكتسبة، المعدل، النقاط) كنقطة انطلاق.
- مقررات الفصل الحالي (المسجّلة بلا تقدير) لإدخال الدرجة/التقدير المتوقع.

ثم يحسب المعدل الفصلي والتراكمي المتوقع لحظياً، ويتيح **طباعة تقرير معتمد**.

> 🔒 **الخصوصية:** تتم كل المعالجة محلياً داخل متصفحك عبر مكتبة `pdf.js`.
> لا يتم رفع الملف ولا حفظ أي بيانات للطالب على أي خادم.

---

## المزايا

- 📄 رفع السجل الأكاديمي PDF (سحب وإفلات) واستخلاص تلقائي — **يدعم السجلات العربية والإنجليزية** (كشف لغة تلقائي).
- ✍️ مراجعة وتعديل كل البيانات المستخلصة قبل الحساب.
- 🧮 سلّمان: **من 5.00** (المعتمد) و**من 4.00**.
- ➕ إضافة/حذف مقررات يدوياً، أو البدء يدوياً بالكامل بدون ملف.
- 🖨️ تقرير معتمد قابل للطباعة (PDF عبر طباعة المتصفح).
- 📱 تصميم متجاوب عربي (RTL) احترافي.

---

## التشغيل محلياً

التطبيق ملفات ثابتة (Static)، لكنه يستخدم وحدات ES و`pdf.js`، لذا **يجب تشغيله عبر خادم محلي**
(فتح `index.html` مباشرة بنظام `file://` قد يمنع تحميل الوحدات والعامل):

```bash
# أي خادم ثابت يكفي، مثلاً:
python -m http.server 8080
# ثم افتح: http://localhost:8080
```

أو عبر إضافة "Live Server" في محرر الأكواد.

## النشر على GitHub Pages

1. أنشئ مستودعاً جديداً على GitHub وارفع محتويات هذا المجلد (عدا مجلد `sample/`).
   ```bash
   git init
   git add .
   git commit -m "حاسبة المعدل - جامعة الملك خالد"
   git branch -M main
   git remote add origin https://github.com/<USERNAME>/<REPO>.git
   git push -u origin main
   ```
2. من إعدادات المستودع: **Settings → Pages**.
3. اختر **Source: Deploy from a branch**، والفرع **main** والمجلد **/(root)**.
4. احفظ، وانتظر دقيقة، وسيظهر رابط الموقع: `https://<USERNAME>.github.io/<REPO>/`.

> لا حاجة لأي خطوة بناء (Build). التطبيق يعمل مباشرة.

  السجلات قليلاً؛ لذا **راجِع البيانات المستخلصة دائماً** وعدّلها قبل الاعتماد عليها.
